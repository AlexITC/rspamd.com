Performance

· Rspamd is designed to work as fast as possible.

· Rspamd can save your computational resources by applying clever techniques, such as event based processing model, abstract syntax tree constructions, carefull algorithms selection and number of global and local optimizations, such as using of hyperscan for optimizing regular expressions.

· Rspamd core is written completely in C programming language and the most critical parts are written in dedicated assembly for the target hardware platforms.

Features

· Rspamd ships with the wide selection of filters to process messages, such as regular expressions, DNS black and white lists, URL black lists, dynamic IP/hosts or DNS lists, SPF module, DKIM plugin and DMARC policy check support.

· For advanced filtering rspamd provides improved statistics module (based on OSB-Bayes algorithm) and fuzzy hashes database that is generated based on honeypots traffic.

· Rspamd also keeps partial compatibility with spamassassin rules via the translation module

Easy to manage

· Rspamd is easy to integrate into the existing e-mail infrastructure and it shipped with the powerful tools to simplify management. For example, the complementary tool rmilter utility provides postfix integration as well as many other features, such as greylisting, ratelimits or clamav checks.

· Rspamd is also integrated with the powerful web interface that simplifies the most common operations and displays statistics.

· Moreover, it is possible to write own rules for rspamd using the marvellous and simple LUA language.
