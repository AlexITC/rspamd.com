Performance

• Rspamd is the mail filtering tool that is designed to work as fast as possible.

• It can save your hardware resources by applying clever techniques, such as event based processing model, abstract syntax tree constructions, carefull algorithms selection and number of global and local optimizations, such as using of hyperscan for optimizing regular expressions.

• Rspamd core is written completely in C programming language and the most critical parts are written in dedicated assembly for the target hardware platforms.

Features

• Rspamd ships with the wide selection of filters to process messages, such as regular expressions, DNS black and white lists, URL black lists, dynamic IP/hosts or DNS lists, SPF module, DKIM plugin and DMARC policy check support.

• For advanced filtering rspamd provides improved statistics module (based on OSB-Bayes algorithm) and fuzzy hashes database that is generated based on spamassassin rules via the translation module

Easy to manage

• Rmilter is a powerfull tool that provides postfix integration as well as many other features, such as greylisting, ratelimits or clamav checks.

• There is also the nice web interface shipped in the rspamd distirbution that simplifies the most common operations and displays statistics.

• Moreover, it is possible to write own rules and plugins for rspamd using the marvellous and simple LUA language.
